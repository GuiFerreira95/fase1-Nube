# fase1-Nube

1️⃣ Nome do projeto
# Vehicle Sales System

2️⃣ Descrição do projeto

Sistema de gerenciamento e venda de veículos, desenvolvido como parte do processo seletivo da Nube. Permite cadastro de veículos, gerenciamento de usuários e simulação de vendas.

3️⃣ Tecnologias utilizadas

Liste as ferramentas, frameworks e linguagens usadas:

## Tecnologias
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- h2
- Maven
- Git/GitHub

4️⃣ Funcionalidades

O que o projeto consegue fazer:

## Funcionalidades
- Cadastro de veículos
- Consulta e listagem de veículos
- Cadastro de usuários e controle de acesso
- Validação de dados
- Integração com banco de dados

5️⃣ Estrutura do projeto

## Estrutura de pastas

src/main/resources
- application.properties
